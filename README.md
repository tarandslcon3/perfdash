**Run API load tests in seconds. Get clear answers instantly. No infrastructure required.**

---

## ⚡ The Problem

API performance testing today is painful:

- ❌ Complex tools like JMeter, k6, Locust
- ❌ Requires scripting, setup, and infrastructure
- ❌ Hard to interpret metrics (avg vs p95 vs spikes)
- ❌ Test data and payloads stored (security risk)
- ❌ Overkill for quick debugging or validation

**You just want a quick answer:**
> “Can my API handle this load or not?”

---

## ✅ The Solution — PerfDash

**PerfDash is a stateless, browser-based API performance testing tool**  
built for developers who need **fast, simple, and secure insights**.

- ⚡ No installation  
- 🧠 No learning curve  
- 🔒 No data storage  
- 📊 Instant results  

---

## 🧪 How It Works (Under 60 Seconds)

1. **Configure Endpoints**  
   Add API URL, method, headers, body — no scripts needed

2. **Set Load & Thresholds**  
   Define virtual users, ramp-up time, duration, and limits

3. **Run Test**  
   Execute instantly and view real-time results

4. **Download Report**  
   Export a clean HTML report with metrics + PASS/FAIL verdict

---

## 📊 What You Get

- ✅ Average latency, p95, p99
- ✅ Requests per second (throughput)
- ✅ Error rate tracking
- ✅ Response time over time
- ✅ PASS/FAIL based on thresholds
- ✅ Downloadable HTML report (shareable)
